# Code of Conduct

## 1. Purpose

This repository is maintained as part of the **academic coursework and laboratory practice** for **Pandit Deendayal Energy University (PDEU)**.  
It aims to promote **ethical learning, academic integrity, collaboration and professionalism** in the study of **Computer Science & Engineering**.

This Code of Conduct establishes clear expectations for behavior to ensure a respectful, inclusive and academically honest environment for all participants.

---

## 2. Our Commitment

As contributors, students and maintainers, we commit to:

- Upholding **PDEU’s academic values and code of ethics**
- Maintaining a **professional and respectful learning environment**
- Encouraging **conceptual understanding over rote copying**
- Supporting open, constructive and respectful collaboration

Participation in this repository implies agreement with this Code of Conduct.

---

## 3. Expected Standards of Behavior

All participants are expected to:

- Communicate respectfully and professionally
- Provide constructive feedback and academic discussions
- Respect diverse perspectives and levels of expertise
- Follow proper coding standards and documentation practices
- Acknowledge and cite external resources where applicable
- Use this repository primarily for **learning, reference and improvement**

---

## 4. Unacceptable Behavior

The following behaviors are strictly prohibited:

- Harassment, discrimination or offensive remarks of any kind
- Personal attacks, abusive language or intimidation
- Plagiarism or misrepresentation of others’ work
- Uploading copied solutions without understanding or attribution
- Any activity that violates **PDEU’s academic integrity policies**
- Misuse of the repository for non-academic or misleading purposes

---

## 5. Academic Integrity and Ethics (PDEU Specific)

This repository is aligned with **PDEU’s Academic Regulations and Examination Ethics**.

- Code is provided **for educational reference only**
- Students must **understand and modify** code before academic submission
- Direct submission of repository code as coursework without comprehension may be considered **academic misconduct**
- Contributors must clearly indicate if code is adapted from textbooks, online sources or peers

Violation of academic integrity principles may lead to reporting under **PDEU disciplinary procedures**.

---

## 6. Contribution Responsibilities

Contributors should ensure that:

- Code is readable, modular and well-commented
- Codes are implemented clearly and correctly
- File names and folder structures follow repository guidelines
- Commit messages are meaningful and professional
- Pull requests include proper explanations and testing where applicable

Maintainers reserve the right to review, modify or reject contributions that do not meet academic or professional standards.

---

## 7. Scope of Application

This Code of Conduct applies to all interactions within this repository, including but not limited to:

- Source code contributions
- Issues, pull requests and discussions
- Comments and documentation

---

## 8. Enforcement

Failure to comply with this Code of Conduct may result in:

- Removal of content or comments
- Rejection of pull requests
- Temporary or permanent restriction from contributing
- Escalation in accordance with **PDEU academic policies**, if required

All enforcement actions will be handled objectively, respectfully and fairly.

---

## 9. Reporting Concerns

Any behavior that violates this Code of Conduct should be reported responsibly by:

- Opening a repository issue, or
- Contacting the repository maintainer privately

All reports will be reviewed confidentially and handled with discretion.

---

## 10. Acknowledgement

This Code of Conduct reflects the principles of **open-source professionalism**, **academic honesty** and **institutional discipline**, aligned with the educational mission of **Pandit Deendayal Energy University (PDEU)**.

---

**Let us uphold integrity, professionalism and excellence in learning.**
